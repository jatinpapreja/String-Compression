# String-Compression

Take as input S, a string. Write a function that does basic string compression. Print the value returned. E.g. for input “aaabbccds” print out a3b2c2d1s1.

Input Format:
A single String S

Constraints:
1 < = length of String < = 100

Output Format
The compressed String.

Sample Input
aaabbccds

Sample Output
a3b2c2d1s1
